## 首屏文案对照

允许文案来自 `docs/desktop-design-system.md`。实装没有新增营销标题、眉题、假指标、头像或搜索框。以下为有意增加的功能性文案：

- `启动开发核心` / `停止开发核心`：用户请求的 Mihomo 生命周期控制；只作用于 36666。
- `出口状态`：为数据表补充可访问区域标题。
- `安全开发模式 / 6666 只读保护`：侧栏安全边界提示。
- `Guardian 已记录`：替代概念图中不准确的“运行中”，避免在没有数据时误报。

## 交互验证

| 路径 | 结果 |
|---|---|
| 启动开发核心 → 路由状态更新 → 停止开发核心 | 浏览器预览状态可往返；真实 Rust 隔离测试也通过 |
| 优先级 → 最低延迟 | Radio 状态和“待三出口接入”说明同步更新 |
| 总览 → 历史 → 返回总览 | 导航和返回操作可用 |
| 立即检测 | Tauri 环境调用 Rust Guardian 并写入 SQLite；浏览器环境使用可辨识的预览数据 |

## 有意偏差与未完成项

- 概念图把生产入口一处错误写成 `16666`，并把 SpeedCat 写成 `16667`；实装按架构事实改为 `6666` 和 `26666`。
- Tauri 使用系统原生窗口边框，没有复刻概念图中的假窗口控制按钮。
- 历史和设置页当前是可导航占位页；完整筛选、CSV、订阅管理和设置表单仍属于后续阶段。
- 当前只接入超实惠 `16666`；订阅 A、SpeedCat `26666` 和自动切换策略没有伪装成已完成。

在上述事实纠正和明确未完成范围内，主总览实现已按概念图完成原尺寸视觉核验，没有剩余可修复的主屏布局问题。
