# 贡献指南

## 提交兼容性报告

请提供以下信息，但务必删除订阅 URL、账号、Token、节点地址、日志中的认证头和本机用户名：

| 字段 | 示例 |
|---|---|
| Windows 版本 | Windows 11 24H2 |
| 客户端版本 | 供应商客户端 3.x |
| Mixed Port 是否可改 | 是 / 否 |
| SOCKS5 TCP | 通过 / 失败 |
| SOCKS5 UDP | 通过 / 失败 / 未验证 |
| 能否并行运行 | 是 / 否 |
| 异常现象 | 端口冲突、路由循环、进程退出等 |

## 代码变更

1. 先创建 Issue，说明要解决的问题和验收方式。
2. 不得提交真实订阅、账号、节点、Token、数据库或运行日志。
3. 涉及代理所有权、Fail Closed 或遥测范围时，同时提交 ADR。
4. Pull Request 中写明测试环境、执行过的检查和回滚方式。
5. 第三方代码必须说明来源、版本和许可证。
