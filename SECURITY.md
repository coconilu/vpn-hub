# 安全策略

## 敏感信息

真实订阅 URL、访问令牌、代理认证、节点地址和运行数据库都不得提交到仓库。正式实现应使用 Windows Credential Manager 或 DPAPI 保存凭据，运行时配置文件仅允许当前用户读取，并对日志进行脱敏。

## 网络默认值

- Mixed Port 和控制 API 默认只监听 `127.0.0.1`。
- 控制 API 必须启用随机 Secret。
- 默认关闭 `allow-lan`。
- 所有出口失效时不得回落到 `DIRECT`。
- 不采集域名、URL、目标地址或流量正文。

## 报告漏洞

请使用 GitHub 的私密漏洞报告功能，不要在公开 Issue 中披露可利用细节、真实凭据或用户数据。若该功能暂不可用，请先提交不含细节的维护者联系请求。
